#6-2  top#  실시간 프로세스 모니터링
`top`              실시간 모니터링 시작
`q`                종료
`k`                PID 입력 후 프로세스 종료
`M`                메모리 사용량 순 정렬
`P`                CPU 사용량 순 정렬
`1`                CPU 코어별 사용률 토글

`htop`             더 보기 좋은 top (별도 설치)
`apt install htop`
`htop` 에서 `F6` 정렬, `F9` kill, `F10` 종료
~강추: htop~   설치만 하면 top 대신 쓸 이유 충분
