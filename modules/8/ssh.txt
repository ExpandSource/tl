#8-3  ssh#  원격 서버 접속
`ssh user@192.168.1.10`        IP로 접속
`ssh user@example.com`         도메인으로 접속
`ssh -p 2222 user@host`        포트 지정
`ssh -i ~/.ssh/key.pem user@host`  키 파일 지정
`exit` or `Ctrl+D`             접속 종료

#~/.ssh/config 설정#
```
Host myserver
  HostName 192.168.1.10
  User alice
  Port 2222
```
→ `ssh myserver` 만으로 접속 가능
