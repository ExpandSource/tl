#3-2  less#  페이지 단위로 파일 보기
`less file.txt`        파일 열기
`q`                    종료
`Space` / `b`          다음 / 이전 페이지
`↑` `↓`               줄 단위 이동
`/keyword`             앞으로 검색
`?keyword`             뒤로 검색
`n` / `N`              다음 / 이전 검색 결과
`g` / `G`              파일 처음 / 끝
`less +F file.txt`     실시간 추적 (로그 볼 때)
~추천: tl log~   로그 파일 볼 때 `less +F` 자주 씀
