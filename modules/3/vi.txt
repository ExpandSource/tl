#3-5  vi#  vi/vim 기본 조작
`vi file.txt`      파일 열기

#모드 전환#
`i`      노멀 → 입력 모드 (편집 시작)
`Esc`    입력 → 노멀 모드 (항상 탈출)

#노멀 모드 명령#
`:w`     저장
`:q`     종료 (변경 없을 때)
`:wq`    저장 후 종료
`:q!`    저장 없이 강제 종료
`u`      실행 취소 (undo)
`Ctrl+R` 다시 실행 (redo)
