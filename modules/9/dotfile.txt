#9-1  dotfile#  설정 파일 이해하기
dotfile = 이름이 `.` 으로 시작하는 설정 파일.
`ls -a ~` 로 확인 가능.

#주요 dotfile#
`~/.bashrc`       bash 시작 시 실행 (alias, 환경변수)
`~/.zshrc`        zsh 시작 시 실행
`~/.profile`      로그인 시 실행 (bash/zsh 공통)
`~/.vimrc`        vim 설정
`~/.gitconfig`    git 전역 설정
`~/.ssh/config`   SSH 호스트 설정

`source ~/.zshrc`  변경 후 즉시 반영 (재시작 없이)

~환경변수: tl env~   $PATH 이해는 필수
