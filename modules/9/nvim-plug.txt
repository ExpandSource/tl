#9-11  nvim-plug#  Neovim 플러그인 관리 (lazy.nvim)
#lazy.nvim 설치#
`~/.config/nvim/init.lua` 에 추가:
```lua
local lazypath = vim.fn.stdpath("data") .. "/lazy/lazy.nvim"
if not vim.loop.fs_stat(lazypath) then
  vim.fn.system({"git","clone","--filter=blob:none",
    "https://github.com/folke/lazy.nvim.git", lazypath})
end
vim.opt.rtp:prepend(lazypath)
```

#추천 플러그인#
`nvim-treesitter`  문법 강조 (정확하고 빠름)
`telescope.nvim`   파일·텍스트 퍼지 검색 (fzf 같은 느낌)
`nvim-lspconfig`   LSP 설정 (자동완성·에러 표시)
`conform.nvim`     자동 포맷터
`catppuccin`       예쁜 컬러 테마

`:Lazy`            플러그인 관리 UI 열기
`:Lazy update`     전체 업데이트
